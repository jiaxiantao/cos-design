## 这是一个 React Hooks 的组件库

#### usage

```bash
yarn add cos-design
# or
# npm install cos-design
```

```bash
import { components } from 'cos-design';
```

#### Start a dev process

```bash
yarn start
# or
# npm start
```

#### Build production bundle

```bash
yarn build
# or
# npm run build
```
#### Build production module

```bash
yarn build:module
# or
# npm run build:module
```
